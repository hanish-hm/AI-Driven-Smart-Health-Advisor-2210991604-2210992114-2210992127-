## 📖 Project Description

The **AI-Driven Smart Health Advisor** is an AI-assisted preventive healthcare web application that provides early-stage health guidance based on user inputs.

The system integrates multiple healthcare-support functionalities such as:

* Vital sign risk assessment (blood pressure, glucose, BMI)
* Symptom-based urgency detection
* Health guideline retrieval
* Outbreak awareness using WHO data
* Nearby healthcare facility suggestions

It acts as a **decision-support tool** to help users understand potential risks and take appropriate actions, but it is not intended for medical diagnosis.

## 🚀 How the System Works

1. User enters vitals, symptoms, and location
2. Backend processes data using FastAPI
3. Risk engine evaluates health conditions
4. Guideline engine retrieves relevant advice
5. Outbreak engine fetches WHO alerts
6. Facility engine suggests nearby healthcare services
7. Results are displayed on the frontend
